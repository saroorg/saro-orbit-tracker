SARO Orbit Tracker

SARO Orbit Tracker is the first public project of SARO — Space Application and Research Organisation.

Features

- Real-time ISS tracking
- Interactive Earth map
- Satellite position monitoring
- Aerospace-inspired dashboard
- Educational space research platform

Roadmap

Version 1.0

- ISS tracking
- Live map
- SARO branding

Version 2.0

- Multiple satellites
- Orbital predictions
- Space weather integration

Version 3.0

- Astra AI assistant
- Mission control center
- CubeSat simulation
